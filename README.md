# JavaScript Todo List App

A simple and beginner-friendly Todo List application built with vanilla JavaScript. Users can add tasks, assign due dates, delete tasks, and their data is saved in the browser using localStorage.

##  Features
- Add new tasks
- Set due dates
- Delete tasks
- Persistent storage with localStorage
- Clean and simple user interface

##  Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

##  What I Learned
- DOM manipulation
- Handling form inputs and events
- Working with arrays and objects
- Saving and retrieving data using localStorage
- Rendering dynamic content with JavaScript
